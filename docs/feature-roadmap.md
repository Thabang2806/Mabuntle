# Swyftly Feature Roadmap

## Foundation

- Monorepo structure.
- Backend solution and project references.
- Angular SSR app shell.
- PostgreSQL compose setup.
- Health check endpoint.

## MVP Sequence

1. Operating rules and marketplace policies.
2. Database schema.
3. Auth, roles, buyer, seller, and admin foundations.
4. Seller onboarding and storefronts.
5. Catalog, products, variants, images, and inventory.
6. Cart and single-seller checkout.
7. Payments, webhooks, ledger, and seller balances.
8. Orders, manual shipping, returns, refunds, and disputes.
9. Admin approval, moderation, support, and reports.
10. AI listing assistant and basic AI moderation.
11. Search, filters, analytics, and seller dashboards.

## Deferred

- Multi-seller checkout.
- Native mobile app.
- Virtual try-on.
- Advanced personalization.
- Complex seller subscriptions.
